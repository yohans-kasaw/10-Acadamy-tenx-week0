# Test and Benchmarking Project

This project serves as a testing environment for agent development and instruction-following benchmarks.

## Directory Structure
- `src/`
  - `packages/`
    - `tests/`
      - `test_sample.py`

## Requirements
- Use Poetry for dependency management.
- Follow strict typing with mypy.
- Adhere to PEP 8 style guide.
- Use Pydantic for data models.
- Implement tests using pytest.

## Setup Instructions
1. Install dependencies using Poetry.
2. Run tests with pytest.
3. Ensure code quality with ruff.
